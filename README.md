# Oponto
App para registro de ponto, possibilitando o funcionario realizar suas marcações de ponto online.

Principais funções do app.
- Registro do ponto
- Solicitação de:
  - Inclusão de registro
  - Horas Extras
  - Banco de Horas
  - Atestados e Justificativas
  - Folgas
  - Ferias
  - Reembolso
